# City-Development-Index-Datesets
## Final Year Project
### Project Title
Deep Learning Architectures for Enhancing the City Development Index and Urban Prosperity Monitoring
## Objective
This repository contains the datasets selected for our Final Year Project. The datasets have been chosen to support a multimodal deep learning approach for evaluating urban development using socioeconomic, environmental, satellite, and infrastructure data.
## Selected Dataset Categories
- 📊 Socioeconomic Data
- 🛰️ Satellite Imagery
- 🌍 GIS / Infrastructure Data
- 🌫️ Air Quality Data
- 🏙️ Quality of Life Indicators
